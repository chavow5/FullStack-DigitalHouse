Funciones = es un listado de procedimientos que se ejecutaran cuando sea necesario

Funcion-expresada =  es aquella que se le asigna como valor una variable 
                   $ let sumar = funtion(){...}
                    y se carga antes de que cualquier codigo sea ejecutado

funcion-declarada = es aquella que recibe un nombre formal y no se asigna como valor a una variable
                    $ function sumar(x,y){}
                    se carga unicamente cuando alcanza la linea donde este la funcion
